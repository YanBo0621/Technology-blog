比较常用的 10 个 markdown 标签

2019/04/24 | markdown

MarkDown 是一种非常轻量级的语言，使用起来也非常方便。可以用它来写文章，做笔
记。目前很多网站都支持 MarkDown 来写博客，这篇文章主要介绍一下它简单常用的标
记。它还有很多更深入的部分，比如流程图、数学公式等，在此不做探讨。

1.标题

1~6 个#[空格]

例如：输入以下标记

一级标题

二级标题

三级标题

四级标题

五级标题
六级标题

返回以下结果：

一级标题

二级标题

三级标题

四级标题

五级标题
六级标题

2.无序列表

格式：*[空格]或者+[空格]或者-[空格]

例如：输入以下标记

  • 蕲春
  • 襄樊
  • 上海

返回以下结果：

  • 蕲春

  • 襄樊

  • 上海

3.有序列表

格式：数字序号.[空格]

例如：输入以下标记

1.苏州

2.杭州

3.丽江

返回以下结果:

1.苏州

2.杭州

3.丽江

4.引用

格式：>空格

例如：输入以下标记

这里是一个引用

返回以下结果:

这里是一个引用

5.插入文字链接

格式:文字

例如：输入以下标记

这是一个链接

返回以下结果:

这是一个链接

6.插入图片

格式:

例如：输入以下标记

返回以下结果:

7.粗体字

格式：两个*或者两个_

例如：输入以下标记

这是粗体字

这是粗体字

返回以下结果:

这是粗体字

这是粗体字

8.斜体字

格式：一个*或者一个_

例如,输入以下标记

这是斜体字

这是斜体字

返回以下结果:

这是斜体字

这是斜体字

9.转义

格式：[字符]

例如,输入以下标记

**这是粗体字**

返回以下结果:

这是粗体字

10.代码块

格式：tab 或 4 个空格

例如,输入以下标记

function log() {

alert(“这是什么？”)

}

返回以下结果：

function log() {
alert(“这是什么？”)
}

注意：各个标记间可以混合使用

例如：
这是一个粗斜体

将返回以下结果：
这是一个粗斜体
